# Course Graph Notebook

The code in this directory show a simple curve plotting example from the course

## Install and Run the Example

- Use the following command to activate the virtual env: ```pipenv shell ```

- In order to install the associated packages on the virtual env, run: ```pipenv install```

- Finally, run the following to start your notebook:``` jupyter-notebook```


- Once started, you should open ``ml_graphics_course.ipynb`` to test the example.

P.S.: ``pipenv`` utilities can be install with ```pip install --user pipenv ```. More information about ``pipenv`` can be found [here](https://pipenv.pypa.io/en/latest/).
